A cor mostra se foi aprovada (verde) ou reprovada (vermelha).

Uma tabela formatada com:
Nome da matéria
Média final
Status (aprovada ou reprovada)


componestes

ft.PieChart (da biblioteca flet_charts) — que é a forma atual de fazer gráficos no Flet.
A tabela foi feita com ft.Row e ft.Text para organizar os dados.

SQLite

banco foi feito na propria main

uma tabela materias com os campos:

Notas (em formato texto, tipo lista)
Média
Status

Ferramentas e bibliotecas usadas
flet: framework para criar interfaces visuais com Python.
sqlite3: banco de dados leve e embutido no próprio Python.
flet_charts: biblioteca para gráficos (gráfico de pizza no seu caso).
